Random Cats API Documentation

Overview

This API allows you to retrieve random cat data. You can specify the number of cats you want to receive and view the documentation for more details.

Endpoints

/api/random-cats

Method: GET

Description: Retrieves random cat data.

Parameters:

  • cats_amount=int - Optional number of random cats to retrieve. (e.g., 1, 2, 3)

Example Request:

GET /api/random-cats?cats_amount=3
